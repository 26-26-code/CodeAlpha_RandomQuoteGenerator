# CodeAlpha_RandomQuoteGenerator

A clean and elegant web app that displays random inspirational quotes.

## ✨ Features
- Displays a random quote every time the page loads
- "New Quote" button to get a different quote instantly
- Shows quote text, author name, and category
- Smooth fade animation between quotes
- Minimal and beautiful dark-themed UI

## 🛠️ Built With
- HTML5
- CSS3
- Vanilla JavaScript

## 🚀 How to Run
1. Download `index.html`
2. Open it in any web browser — no installation needed!
